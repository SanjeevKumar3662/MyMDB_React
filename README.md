# Sanjeev's MDB

This site is indexed on google, search **"sanjeevsmdb"**
Link : [https://mdb.sanjeevkumar.site/] New custom domain
Link : [https://sanjeevsmdb.vercel.app/] old url, still active

### Notes :

- This is a Movie Database Website
- Uses TMDB's API for movie/ tv data
- Uses JWT for auth
- LoggedIn users can maintain a watchlist and select status of media from the following options -
  watching
  plan to watch
  completed
  on hold
  dropped
